本项目涉及到的技术栈如下

## 后端

### 核心框架

Java SpringBoot：项目核心框架，用于快速构建Java后端应用

### 数据库相关

- MySQL：作为后端对接的数据库
- MyBatis：用于Java与MySQL数据库交互的持久层框架

### 开发 / 辅助工具

- Maven：项目构建工具
- Navicat：数据库可视化管理工具
